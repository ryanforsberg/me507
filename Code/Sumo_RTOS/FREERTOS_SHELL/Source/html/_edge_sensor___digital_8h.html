Header for the Edge Detection task.

The task_Motor class sets up the XMEGA PWM for 1kHz operation on each of the four motor driver channels. Each motor driver will be driven at the same duty cycle. Based on the four input booleans and one uint8_t, the task will set the outputs according to the following truth table. NOTE! Back Left is interpreted as turning to FACE left, which will result in the chassis moving to the RIGHT. PS - I can't get Doxygen tables to work, please see the source code.
